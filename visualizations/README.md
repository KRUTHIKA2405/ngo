# NGO CSR Funding Analysis - Visualizations

All visualizations have been generated as PNG files in this folder. You can view them directly by opening the PNG files in your image viewer or file explorer.

## Available Visualizations:

1. **xgboost_classification.png** - Confusion Matrix and Feature Importance for XGBoost classifier
2. **donor_probability_analysis.png** - Top 15 potential donors and probability distribution
3. **dbscan_clustering.png** - DBSCAN clustering results with PCA visualization and probability coloring
4. **regression_analysis.png** - Actual vs predicted education spending and feature importance
5. **model_performance_metrics.png** - ROC curve, probability distributions, calibration curve, and learning curve
6. **3d_clustering.png** - 3D scatter plot of DBSCAN clusters
7. **cluster_boxplots.png** - Box plots comparing CSR Spend, Education Spend, and Growth across clusters
8. **correlation_heatmap.png** - Correlation matrix of all features

## How to View:
- Open the PNG files directly in your image viewer
- Or use the file explorer to browse and open them
- Each visualization is saved at high resolution (300 DPI) for clear viewing

- XGBoost achieved 100% accuracy in predicting high-funding donors
- DBSCAN identified 4 clusters of companies
- Regression model predicts education spending with R² = 0.9910
